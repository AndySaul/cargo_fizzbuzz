# cargo_fizzbuzz

Rust deliberate practice using Cargo features

Code is separated into different files for app, lib and tests.

Initial commit created with
```rust
cargo new fizzbuzz -lib
```

https://doc.rust-lang.org/cargo/index.html

Build crate:
```rust
cargo build --release
```

Test lib (& doc tests):
```rust
cargo test
```

Generate rust docs:
```rust
cargo doc
```

Run the app
```rust
cargo run
```
